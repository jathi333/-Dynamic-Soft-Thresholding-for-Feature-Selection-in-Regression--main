# Numerical Optimization Project

## Project Title
Dynamic Soft-Thresholding for Feature Selection in Regression

## Description
This project applies numerical optimization techniques to regression models
for predicting housing prices. The project compares Linear Regression,
Ridge Regression, LASSO Regression, and Dynamic LASSO optimization.

## Dataset
Housing price dataset containing 545 samples with 13 features.

## Project Structure

dataset/
    housing.csv

code/
    model.py

graph/
    model_comparison.png

main.py
requirements.txt

## Installation

Install required libraries:

pip install -r requirements.txt

## Running the Project

Run the project using:

python main.py

## Output

The program will:

- Train regression models
- Compare model performance
- Generate a comparison graph

The graph will be saved in:

graph/model_comparison.png
